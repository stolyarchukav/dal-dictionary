##### Android application 
###### Explanatory Dictionary of the Live Great Russian language of Vladimir Ivanovich Dal
##### Features
* Words search
* Widget
* Embedded indexed database
* Bookmarks
* Search history

##### Build command
~~~
gradlew clean build
~~~
